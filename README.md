# Stellar Burgers: автоматизация юнит-тестирования

### Описание проекта

В рамках проекта были разработаны юнит-тесты для приложения космической бургерной **Stellar Burgers**.

### Цель проекта

Проверить корректность работы основных компонентов приложения с помощью модульных тестов и обеспечить высокое покрытие кода.

### Реализованные сценарии

Созданы юнит-тесты для классов:
`Bun`, `Burger`, `Database`, `Ingredient`

Покрытие кода: **100%**  
HTML-отчет формируется в директории `htmlcov/` и доступен в файле `index.html`

### Использованные подходы

В проекте применены:
- **фикстуры Pytest** для создания тестовых объектов и подготовки тестовых данных;
- **Mock-объекты** для изоляции зависимостей класса `Burger`;
- **параметризация тестов** для проверки различных сценариев расчета цены и формирования чека.

### Технологии

Python 3.14.6, Pytest, pytest-cov, unittest.mock

### Структура проекта

- `praktikum` — исходный код приложения;
- `tests` — автоматизированные тесты;
- `conftest.py` — фикстуры;
- `data.py` — тестовые данные;
- `README.md` — описание проекта;
- `requirements.txt` — внешние зависимости.

### Результат

В проекте реализован полный набор юнит-тестов для ключевой бизнес-логики приложения Stellar Burgers.

Достигнуто:
- 100% покрытие тестами;
- изолированное тестирование компонентов;
- проверка бизнес‑логики формирования бургера, расчета цены и генерации чека.

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `pytest --cov=praktikum --cov-report=html`
